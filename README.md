# Budget friendly battery sag pedal
im working on making a more budget friendly option for a battery sag pedal and hopefully others in the future.
The motivation for this project is my dad, he came to me one day telling me about battery sag pedalds (for context we both play guitar) and he was telling me that they are over 100 dollars on amazon but they arent very complicated so i decided i wanted to make a cheaper version. So now I'm working on designing my own version for cheap. Im also hoping to make more cheaper versions of pedals.
